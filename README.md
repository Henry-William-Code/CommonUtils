# CommonUtils

#### 介绍
Android中的常用的工具类

#### 软件架构
无


#### 安装教程

无

#### 使用说明
1、在工程中的builder.gradle中添加
    allprojects {
		repositories {
			...
			maven { url 'https://jitpack.io' }
		}
	}
2、在单独项目中的builder.gradle中添加
    dependencies {
	        implementation 'com.github.Henry-William-Code:CommonUtils:版本号'
	}
